A inventory management system built in **Java** with **SQLite** for persistent data storage.  
This project helps manage items, track stock levels, and generate simple reports ideal for students or small businesses.

---

## Features
- Add, update, and remove inventory items  
- View all items, filter by category, or sort by quantity/value  
- Display total inventory worth  
- Identify low-stock items automatically  
- Undo last action (add/remove/update)  
- Import and export inventory as CSV  
- Save and load data automatically with SQLite  
- Ready for future PDF export



## Technologies
- **Language:** Java (JDK 17+)
- **Database:** SQLite (via `sqlite-jdbc`)
- **Environment:** NetBeans IDE
- **Libraries:** JDBC, OpenPDF (optional)

---

## How to Run
1. Clone or download this repository.  
2. Open in **NetBeans**.  
3. Add required libraries:
   - `sqlite-jdbc-3.46.0.0.jar`  
   - *(optional-You Should)* `openpdf-1.3.30.jar`  
4. Run `MainApp.java` from the terminal view.
